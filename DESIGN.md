# DESIGN.md

## 气质与意象

黑暗魔法大厅 — 午夜霍格沃茨大厅，漂浮的蜡烛照亮古老石墙，空气中弥漫着魔药蒸汽。分院帽歪斜地坐在高脚凳上，破旧的帽褶里藏着千年智慧。厚重丝绒帷幕，暗金色烛光在银器上跳动。

## 配色方案

- **主背景**: 深邃午夜蓝 `#0a0e1a` → 暗石板 `#111827` 渐变，模拟大厅穹顶
- **次背景**: 暗紫褐 `#1a1025`，石墙质感
- **主强调色**: 暗金 `#c9a84c`，烛光/魔法光芒
- **辅助强调**: 琥珀 `#d4a017`
- **文字主色**: 羊皮纸白 `#e8dcc8`
- **文字次色**: 灰褐 `#9ca3af`
- **学院色**:
  - 格兰芬多: 深红 `#740001` + 金 `#d3a625`
  - 斯莱特林: 墨绿 `#1a472a` + 银 `#aaaaaa`
  - 拉文克劳: 深蓝 `#0e1a40` + 铜 `#946b2d`
  - 赫奇帕奇: 鹅黄 `#ecb939` + 黑 `#372e29`

## 字体排版

- 标题: Cinzel (Google Fonts)，哥特衬线体，用于学院名、咒语名
- 正文: Noto Serif SC，中文正文阅读
- 咒语: Cinzel Decorative，拉丁咒语专用
- 排版节奏: 大标题大字重，间距宽松，营造庄重感

## 视觉策略

- 背景使用深色渐变 + 微光粒子动画（漂浮尘埃/星尘效果）
- 卡片使用半透明暗色毛玻璃 `rgba(15,15,30,0.8)` + backdrop-blur
- 边框使用暗金色细线 `1px solid rgba(201,168,76,0.3)`
- 按钮使用暗金渐变，hover 发光
- 咒语文字使用金色发光效果 `text-shadow`
- 魔杖轨迹使用发光粒子效果

## 动效与交互

- 页面切换: 淡入淡出 + 轻微缩放，如魔法消散
- 分院帽: 微微摇晃的 CSS 动画
- 音频可视化: 波形脉动，金色调
- 魔杖轨迹: 发光粒子尾迹，金色核心 + 外围光晕
- 分院结果: 大字逐字显现 + 金色粒子爆发
- 按钮 hover: 边框发光 + 内部光晕扩散

## 结果页设计规范（v2）

### 大字报原则
- 所有标题至少 `text-5xl` 起步，正文至少 `text-xl`
- 元素尺寸放大 1.5-2 倍，确保远处观众可清晰辨识
- 每个步骤页面元素精简，单屏完整展示，无需滚动

### 鎏金边框特效
- 使用 CSS `linear-gradient` 模拟鎏金金属光泽
- 边框带有微弱光晕动画（`gildedShine` 关键帧）
- 四角装饰：2px 实色边框缺角，形成画框角效果
- 画框内阴影：使用对应学院色的 glow 光晕

### 牛皮纸质感（评估报告页）
- 背景色：泛黄做旧 `#d4c5a9` → `#c9b896` → `#b8a57c`
- SVG noise 纹理叠加，`mix-blend-mode: multiply`
- 深褐色文字 `#5c4033`，模拟古老墨水
- 火漆印装饰：深红色圆形，内嵌浮雕虚线边框

### 学院画框（巫师肖像页）
- 动态边框颜色：根据学院匹配对应主色 + 辅色
  - 格兰芬多：深红 `#740001` + 金 `#d3a625`
  - 斯莱特林：墨绿 `#1a472a` + 银 `#aaaaaa`
  - 拉文克劳：深蓝 `#0e1a40` + 铜 `#946b2d`
  - 赫奇帕奇：鹅黄 `#ecb939` + 黑 `#372e29`
- 画框内发光阴影：`box-shadow: inset 0 0 60px {glowColor}`
- 四角星芒装饰边框

### 布局策略
- 评估报告：左右双栏，牛皮纸大卡片占满屏幕
- 专属徽章/魔杖：中央大展示 + 鎏金金属卡片 + 两侧信息
- 巫师肖像：左侧超大肖像画框（480x480）+ 右侧二维码（160x160）+ 学院中文名大字

## 设计禁忌

- 禁止使用明亮白色大块背景
- 禁止使用科技蓝/紫色渐变
- 禁止圆角过大的卡片（最大 12px）
- 禁止扁平无质感的按钮
- 禁止现代感无衬线字体做标题
- 禁止结果页元素过小导致远处无法看清
